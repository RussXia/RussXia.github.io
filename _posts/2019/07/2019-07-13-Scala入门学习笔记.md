# Scala基础

## 变量
定义不可变量:val, val类似于Java中的final变量。与之对应的var类似于Java中的非final变量。Scala自带类型推断(type inference),
```scala
scala> val a = 2
a: Int = 2

scala> var b = 3
b: Int = 3

scala> val str:String = "Hello World";
str: String = Hello World

scala> println(str);
Hello World
```
